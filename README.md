### Histórico de Comandos

```
pnpm dlx create-turbo@latest
```

`.\config\prettier`

```
pnpm i prettier -D
```

```
pnpm i prettier-plugin-tailwindcss -D
```

`.\config\eslint-config`

```
pnpm i @rocketseat/eslint-config -D
```

```
pnpm i eslint-plugin-simple-import-sort -D
```

`.\packages\auth`

```
pnpm i @casl/ability
```

`.\apps\api`

```
pnpm i tsx -D
```

```
pnpm i @types/node -D
```
